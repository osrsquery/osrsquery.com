What is OSRSQUERY

A modern Runescape platform dedicated to providing the community an environment that anyone can use to gather data. OSRSQuery hosts a variety of projects with tools like Oldschool Runescape Item, NPC, Model, and Object databases. We also have OSRS Sprites, Worldmap, Textures, Music, and Sounds.

OSRSQUERY features

A collection of online tools for Old School RuneScape enthusiasts, free to use!

Item Search

Search the most up-to-date list of Oldschool Runescape (OSRS) Item ID numbers and Item names. OSRSQuery offers many revisions.

NPC Search

Search the most up-to-date list of Oldschool Runescape (OSRS) NPC ID numbers and NPC names. OSRSQuery offers many revisions.

Model Search

Search the most up-to-date list of Oldschool Runescape (OSRS) Model ID numbers and Model names. OSRSQuery offers many revisions.

Object Search

Search the most up-to-date list of Oldschool Runescape (OSRS) Object ID numbers and Object names. OSRSQuery offers many revisions.

Worldmap

An interactive Oldschool Runescape (OSRS) world map with coordinates, regions, and area / path creation. Updated weekly.
